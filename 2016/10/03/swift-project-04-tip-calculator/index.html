Swift Project 04 - TIP CALCULATOR

发表于 2016-10-03 更新于 2025-02-19 分类于 Development Disqus： 本文字数： 476 阅读时长 ≈ 1 分钟

主要学习的内容：

  • 了解了 Navigation Bar
  • 使用了 MVC 架构（其实一直都是，只是把计算部分独立了出来而已）
  • 了解键盘 Number Pad 和 Decimal Pad 的区别（有无小数点）
  • 给 Keyboard 增加 Done 按键（UIBarButtonItem 组件）
  • 格式化 String
  • NSString 转化成 Float 类型的数据
  • UISlider 组件

开发环境：

  • macOS 10.12
  • Xcode 8.0
  • iOS 10.0
  • Swift 3.0

组件：

  • UIBarButtonItem
  • UISlider
